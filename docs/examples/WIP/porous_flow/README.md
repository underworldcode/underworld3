# Porous Flow - WIP

Advanced porous flow examples that complete the porous flow physics domain.

## Examples

### `Ex_Explicit_Flow_Grains.ipynb`
**Explicit grain-scale flow modeling**

Advanced simulation of fluid flow around individual grains in porous media using explicit particle representations.

**Topics covered**:
- Grain-scale fluid dynamics
- Explicit particle-fluid coupling
- Multi-scale flow modeling

**Prerequisites**: 
- Understanding of fluid mechanics
- Familiarity with particle methods
- Basic porous media concepts

### `pramoda_stokes.ipynb`
**Stokes flow in porous media**

Detailed analysis of Stokes flow through porous structures with complex boundary conditions.

**Topics covered**:
- Stokes flow formulation
- Porous media boundary conditions
- Flow field analysis and visualization

**Prerequisites**:
- Stokes flow fundamentals
- Porous media theory
- Boundary condition implementation
